Lecture — 3D とは？ ①

データになる立体空間

🍉　スイカ割りで考えてみよう
目隠しをしてスイカ割りをするとき、
「右に2歩、前に3歩」と声をかけてもらう。

これは「空間の中の位置」を数字で伝えていること。

コンピュータの中の3D空間も同じ。
どんな点も X・Y・Z の3つの数字 で表せる。
点（バーテックス）
座標 (X, Y, Z) ひとつの位置
面（ポリゴン）
点3つ以上が作る平面の形
立体（3D メッシュ）
面が集まって立体になる！

Lecture — 3D とは？ ②

空間の表現方法はひとつじゃない

🔷
ポリゴンメッシュ
POLYGON MESH
点・線・面で立体を表す。今日使う方法。ゲーム・映像・アニメーションに広く使われる。
→ Nomad Sculpt、Blender、Maya など
🔵
点群
POINT CLOUD
空間をびっしりと並んだ点の集合で表す。3Dスキャナーや LiDAR が得意とする形式。
→ 建築測量・自動運転・文化財保存など
🫧
ガウシアンスプラッティング
GAUSSIAN SPLATTING
点群を発展させた技術。ふわっとした「ガウス関数の塊（スプラット）」で空間を表現。写真から高精度に再現できる。
→ 2023年以降に急速に普及した新技術
🟥
ボクセル
VOXEL
3Dのマス目（立方体のピクセル）で空間を表す。医療画像（CT・MRI）の可視化などに使われる。
→ 医療・科学シミュレーションなど

Lecture — Modeling

モデリングの種類

スカルプト
SCULPT MODELING
粘土をこねるように自由に形をつくる。有機的なキャラクター・生物・食べ物など、曲線の多い形が得意。
例：キャラ・モンスター・ドーナツ
ポリゴン
POLYGON MODELING
四角や三角の面を一枚ずつ組み合わせて形をつくる。建物・メカ・硬い形が得意。
例：建物・車・武器・家具
プロシージャル
PROCEDURAL
数式やルールで自動的に形を生成する。自然現象・大量の繰り返し形状に強い。
例：炎・水・地形　Houdini が得意
AI
AI MODELING
テキストや画像から AI が形を生成する。2023年以降に急速に発展している新しい手法。
例：Luma AI、TripoSR など

Lecture — Sculpt

スカルプトモデリングとは

スカルプトとは
粘土のように自由に形をつくる手法。ブラシで押したり引いたり盛ったりして、直感的にモデルをつくることができる。
ハイポリ（高ポリゴン）とは
  • 面（ポリゴン）の数がとても多い
  • きめ細かくリアルな形が作れる
  • 計算量が多く、処理が重い
⚠️　こまめに保存しよう！
  • Nomad Sculpt は自動でデータを保存しない
  • アプリを閉じると、保存していない作業は消える
☰ → Files → 上書き保存

マスク → メッシュ分離

チョコ部分を別パーツとして切り出す ── 全員手を止めて見てね！
マスクとは？
  • メッシュの「一部だけ」を選んで操作する機能
  • 塗った部分は動かせなくなる（保護される）
  • 逆に、マスク外だけをスカルプトすることもできる
今回はマスクを「切り出し範囲の指定」として使う
操作手順（Extract）
1
Mask ブラシを選ぶ
ブラシリストから Mask を選択する
2
チョコ部分を塗る
ドーナツの上面をなぞって塗り込む
3
Extract → New Mesh
メニューから Extract を実行して分離する
Extract → New Mesh
4
名前をつけて管理する
Scene パネルでパーツ名を変更する
