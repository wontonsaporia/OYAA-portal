Lecture — 3DCG とは

3DCG ってなんだ？

5
3D
3 Dimension（3次元）
タテ・ヨコ・奥行きの3つの軸をもつ空間のこと。私たちが生きる世界がそのまま3次元です。
CG
Computer Graphics
コンピューターで作った映像・画像のこと。2DCGと3DCGがあります。
＝
コンピューターの中に 3D 空間を再現する技術
映画・ゲーム・建築・医療・広告など、あらゆる分野で使われています。
3DCGでは、この3D空間にポリゴン（三角・四角形の面）を組み合わせてモノの形を作ります。その形に色・質感・光をあてて レンダリング することで、1枚の画像として出力されます。

Lecture — データになる空間

データで表す 3D 空間

6
アナロジー — スイカ割り
目隠しをして、スイカに向かって棒を振る。
「3歩前に進んで、少し左、そこで振り下ろして」——
これは 位置を座標で指示している ことと同じです。

コンピューターの3D空間も同じ仕組みです。空間のすべての点は、3つの数値（x, y, z）で一意に定まります。形・色・光——すべてが 数値のデータ として記録されています。
座標系（COORDINATE SYSTEM）
X
左右（横）
Y
上下（縦）
Z
奥行き
3D空間内のどんな点も、（x, y, z） の3つの数値で表せます。スカルプトでブラシを動かすとき、実際には頂点の座標が書き換わっています。
例：ドーナツの表面の頂点
p₀ = ( 1.20, 0.00, 0.45 )
p₁ = ( 1.18, 0.03, 0.47 )
⋮　← これが集まって「形」になる

Lecture — 空間の表現

3D空間の表し方はひとつじゃない

7
形をどう記録・表現するかの違いが、用途や得意なことの違いにもつながります。
ポリゴンメッシュ
POLYGON MESH
頂点・辺・面の集合体。面を組み合わせて形を表現します。3DCGの主流形式です。
→ 映像・ゲーム・キャラクター全般
今日使う
NURBS
NURBS SURFACE
数式（制御点＋重み）で曲面を定義します。拡大しても真に滑らかな曲面を保ちます。
→ 工業設計・自動車・建築・CAD
ボクセル
VOXEL
3D空間をグリッドに分割し、各セルのON/OFFで形を表現します。内部構造を持てます。
→ 医療CT・地形破壊・AI学習
点群
POINT CLOUD
座標と色を持つ点の集合。面・辺の概念がなく、3Dスキャナーの生データ形式です。
→ 測量・建築・自動運転・LiDAR
SDF
SIGNED DISTANCE FIELD
空間の各点が表面からの距離を関数で表現。形の合体・くり抜きが得意です。
→ AIによる3D生成・リアルタイムCG
3D Gaussian Splatting
GAUSSIAN SPLATTING
点群を発展させ、各点に楕円体の広がりと透明度を付加。写真・動画から高速に3Dを再構成できます。
→ 写真・動画からの3D再構成・AI

Lecture — スカルプト

スカルプトの仕組み

11
スカルプト（彫刻）というその名の通り、すでにあるメッシュに対してブラシで押し出したり盛り上げたりするモデリング手法です。
VERTEX
頂点
x・y・z の座標をもつ点
EDGE
辺
頂点と頂点をつないだ線
FACE
ポリゴン
辺で囲まれた三角・四角形
MESH
メッシュ
ポリゴンが集まった形
ブラシを当てると、円の範囲内にある 頂点の座標（x・y・z） が一斉に指定方向へ移動します。粘土と違うのは、動いているのが「物質」ではなく 「数値」 であることです。
HIGHPOLY
ハイポリ
頂点が多い → 細かく変形できる → 滑らか / 処理が重い
LOWPOLY
ローポリ
頂点が少ない → 大まかにしか変形できない → 軽い / 荒い
リメッシュ — スカルプト中に頂点の密度・構造が崩れたとき、均一に整え直す機能です。Nomad Sculpt では頻繁に使います。
⚠ 保存について
Nomad Sculpt は 自動保存しません。10分に1回、☰ → Files で上書き保存しましょう。
